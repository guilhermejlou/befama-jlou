Funções Exponencial e Logarítmica

50 min Potenciação · Domínio · Logaritmo · Gráficos 4 exercícios de vestibular
Fundamento

1. Potenciação

Antes de entrar nas funções, precisamos dominar a linguagem por trás de tudo.

Definição

aⁿ = 1 × a × a × … × a  (o "1" na frente, multiplicado por a exatamente n vezes)

O 1 inicial deixa claro por que a⁰ = 1: quando multiplicamos zero vezes, só sobra o 1 de partida. Exemplo: 2³ = 1 × 2 × 2 × 2 = 8.

Propriedades essenciais

Produto · mesma base
aᵐ · aⁿ = aᵐ⁺ⁿ
Quociente · mesma base
aᵐ ÷ aⁿ = aᵐ⁻ⁿ
Potência de potência
(aᵐ)ⁿ = aᵐⁿ
Expoente zero
a⁰ = 1, com a ≠ 0
Expoente negativo
a⁻ⁿ = 1 / aⁿ
Expoente fracionário
a^(1/n) = ⁿ√a
✦ Ativação: calcule 2¹⁰ dobrando o valor passo a passo → 2, 4, 8, 16, 32, 64, 128, 256, 512, 1024. Esse crescimento acelerado é a essência da função exponencial.
Conceito central

2. Funções como máquinas

Uma função é uma máquina: você coloca um valor de entrada (input), a máquina processa e devolve um valor de saída (output). Nem todo input é válido — o conjunto de entradas válidas se chama domínio.

Domínio e Imagem

Domínio = conjunto de entradas válidas da função.
Imagem = conjunto de saídas possíveis.

Na máquina de sucos: frutas ∈ domínio (produzem resultado), vegetais ∉ domínio (geram erro).

Função

3. Função Exponencial

Definição

f(x) = aˣ, com a > 0 e a ≠ 1

O expoente é a variável — é isso que diferencia da potenciação comum.

2.0
Base 2 → crescente. A curva tende a 0 quando x→−∞ (assíntota horizontal y=0) mas nunca toca o eixo x.

ℝ (todos os reais)

⚠ Base entre 0 e 1 (ex: 0,5): função decrescente — representa decaimento (radioatividade, depreciação, resfriamento).
Inversa

4. Logaritmo — a regra fundamental

Definição

logb a = x  ⟺  bˣ = a

O logaritmo de a na base b é o expoente x ao qual, na base b, resulta em a.

Condições de existência: a > 0, b > 0, b ≠ 1

✦ Se a potenciação pergunta "qual é o resultado?", o logaritmo pergunta "qual foi o expoente?"

Exemplos guiados

  • log₂ 8 = 3  ← porque 2³ = 8
  • log₁₀ 1000 = 3  ← porque 10³ = 1000
  • log₅ 1 = 0  ← porque 5⁰ = 1
  • log₃ (1/9) = −2  ← porque 3⁻² = 1/9
  • log₄ 2 = 1/2  ← porque 4^(1/2) = √4 = 2

Calculadora interativa

5. Propriedades dos logaritmos

Cada propriedade abaixo nasce diretamente de uma propriedade da potenciação.

1 — Logaritmo do produto
loga(M · N) = loga M + loga N

Ex: log₂(4 · 8) = log₂4 + log₂8 = 2 + 3 = 5

2 — Logaritmo do quociente
loga(M / N) = loga M − loga N

Ex: log₂(32/4) = log₂32 − log₂4 = 5 − 2 = 3

3 — Logaritmo da potência
loga(Mⁿ) = n · loga M

Ex: log₂(8²) = 2 · log₂8 = 2 · 3 = 6

4 — Mudança de base
loga b = logc b / logc a

Permite calcular qualquer log usando log₁₀ ou ln da calculadora.

⚠ Erro clássico: log(M + N) ≠ log M + log N. A propriedade é para produto, não para soma!
✦ Antes das calculadoras, logaritmos transformavam multiplicações em adições — foi o "processador" dos séculos XVII–XIX, usado em astronomia e navegação.
Visualização

6. Os gráficos e a relação entre elas

f(x) = aˣ e g(x) = loga x são funções inversas — seus gráficos são simétricos em relação à reta y = x. Escala 1:1 real abaixo.

2.0
Por que são inversas?

Se (a, b) está em aˣ, então (b, a) está em loga x. Trocar x e y é a operação de inversão — geometricamente, é o reflexo em y = x.

ENEM 201901

A escala Richter mede a intensidade de terremotos com M = log(A/A₀), onde A é a amplitude medida e A₀ é uma amplitude de referência. Se um segundo terremoto tem amplitude 1 000 vezes maior que a de referência, qual é sua magnitude?

  • A1
  • B2
  • C3
  • D4
  • E5
Gabarito: C — Magnitude 3

M = log(1000/1) = log(10³) = 3 · log(10) = 3 · 1 = 3
FUVEST 202202

Sabe-se que log 2 ≈ 0,301 e log 3 ≈ 0,477. Com esses valores, qual é o valor aproximado de log 12?

  • A0,778
  • B0,903
  • C1,000
  • D1,079
  • E1,146
Gabarito: D — 1,079

12 = 2² · 3 → log 12 = 2·log 2 + log 3 = 2×0,301 + 0,477 = 0,602 + 0,477 = 1,079
UNICAMP 202303

Uma colônia de bactérias dobra a cada hora. Se há 500 bactérias no instante t = 0, a quantidade é P(t) = 500 · 2ᵗ. Em quantas horas a população ultrapassa 16 000?

  • A3 horas
  • B4 horas
  • C5 horas
  • D6 horas
  • E8 horas
Gabarito: C — 5 horas

500 · 2ᵗ > 16 000 → 2ᵗ > 32 = 2⁵ → t > 5 → em t = 5: P(5) = 500 · 32 = 16 000 ✓
ENEM 202104

O pH é definido por pH = −log[H⁺]. Uma solução com pH = 5 tem [H⁺] = 10⁻⁵ mol/L. Quantas vezes maior é a concentração de H⁺ de uma solução com pH = 3 em relação à de pH = 5?

  • A2 vezes
  • B10 vezes
  • C50 vezes
  • D100 vezes
  • E1 000 vezes
Gabarito: D — 100 vezes

pH 3 → [H⁺] = 10⁻³  |  pH 5 → [H⁺] = 10⁻⁵
Razão = 10⁻³ / 10⁻⁵ = 10² = 100 vezes
